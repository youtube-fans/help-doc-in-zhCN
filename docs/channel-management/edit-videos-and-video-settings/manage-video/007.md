# 查看视频的观看限制

通过“限制”列，您可以了解观看者观看您的视频时是否受限。如果您是具有获利资格的创作者，则还可以了解视频的获利功能是否受限。您可以在 YouTube Studio 的[视频页面](http://youtube.com/my_videos)上找到“限制”列。

## 查找视频的观看限制

要了解您的视频是否受限，请执行以下操作：

1. 登录 [YouTube Studio](https://studio.youtube.com/)。
2. 在左侧菜单中，选择 **视频** 。
3. 在列表中找到您的视频，然后查看“限制”列。

## 限制类型

#### 版权

如果您收到了 Content ID 版权主张或版权警示，您会在“限制”列中看到相应内容。如果您对某项版权主张提出了异议，还会看到异议状态。例如，如果您向版权所有者提交了申诉，就会看到“申诉中”字样。详细了解 [YouTube 内容版权](https://support.google.com/youtube/topic/2676339)

#### 条款及政策

如果您的视频因使用条款方面的问题而[被移除](https://support.google.com/youtube/answer/6395024)、[受限](https://support.google.com/youtube/answer/7458465)，或者[被 YouTube 设为“私享”](https://support.google.com/youtube/answer/7300965)，您会在“限制”列中看到“使用条款”。例如，如果出现以下情况，您可能会看到使用条款问题：

* 您的视频含有违反[社区准则](https://support.google.com/youtube/answer/9288567)的不当内容
* [人为增加了您的观看次数](https://support.google.com/youtube/answer/3399767)
* 您违反了 [YouTube 服务条款](https://www.youtube.com/static?template=terms)
* 您的视频存在[商标](https://support.google.com/youtube/answer/2801979)方面的问题
* 我们收到了针对您的视频提交的法律投诉

#### 与年龄相关的限制

如果您的视频是面向儿童的内容或仅适合成人，您会在“限制”列中看到一条相关消息。可能会显示以下与年龄相关的限制：

* 面向儿童的内容（由您设置）
* 设置为“面向儿童的内容”（由 YouTube 设置）
* 年龄限制（由您设置）
* 年龄限制（由 YouTube 设置）

详细了解[有年龄限制的内容](https://support.google.com/youtube/answer/2802167)或[面向儿童的内容](https://support.google.com/youtube/answer/9528076)。

#### 是否适合投放广告（仅限符合条件的创作者）

如果您是具有获利资格的创作者，您会在“限制”列中看到一条消息，说明您的视频是否因包含不适合大多数广告客户投放广告的内容而只能投放部分广告或无广告。如果您已申请审核，则还会在“限制”列中看到审核状态。[详细了解获利功能的变化](https://support.google.com/youtube/answer/7561938)。
