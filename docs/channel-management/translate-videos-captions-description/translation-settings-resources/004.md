# 翻译和转录词汇表

* **ASR** ：自动语音识别。YouTube 可使用自动语音识别功能来向视频添加自动字幕（支持英语、荷兰语、法语、德语、意大利语、日语、韩语、葡萄牙语、俄语和西班牙语）。并非所有视频都支持自动语音识别功能。
* **自动字幕** ：由自动语音识别功能创建的字幕轨道。
* **字幕** ：用于指代视频中以文本形式出现的原始语言转录字幕和翻译字幕。默认情况下，“字幕”指的是原始语言转录字幕。
* **隐藏式字幕** ：隐藏式字幕用文字来描述视频中的音频内容。这种字幕主要是为了方便听力有障碍和失聪的观众。内容包括台词转录以及声音提示（例如“[音乐播放]”或“[笑声]”）。隐藏式字幕还可以用来标识说话者，例如“小王：您好！”或在屏幕上指示说话者的位置。
* **投稿** ：创建或修改元数据翻译或新的字幕轨道，以便发布到视频中。
* **投稿内容** ：新建或修改后的元数据翻译、翻译字幕或隐藏式字幕，可在审核后发布到视频中。
* **贡献者** ：提交过新的翻译字幕内容、隐藏式字幕内容或元数据翻译的志愿者；或者修改或审核过其他贡献者的内容的志愿者。
* **创作者** ：视频上传者/所有者。
* **提交** ：将完整或部分完成的字幕轨道送交审核，以便发布到视频中。
* **提交的内容** ：送交审核以供发布到视频中的完整或部分完成的翻译或转录内容。
* **翻译字幕** ：与视频一起显示的文字轨道，采用的语言不同于视频中的对话语言。这种字幕主要为外语观众而准备。其内容是对语音对话和书面文字的翻译，显示在视频底部或下方（“子”标题）。
* **设置显示时间** ：用户提交语音稿后，我们的同步服务器会自动对语音稿和视频进行对齐，从而创建有时间戳的字幕轨道。
* **转录稿** ：无格式（且无时间戳）的文字，与视频中的对话内容逐字对应。
* **翻译** ：通过翻译现有元数据、翻译字幕或隐藏式字幕而制作出的标题、说明或翻译字幕。
