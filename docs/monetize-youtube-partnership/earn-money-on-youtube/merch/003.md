# 商品网站和众筹网站

您可以从您的视频链接至您的商品或众筹网站，前提是这些网站在获准网站之列，并且您已加入 [YouTube 合作伙伴计划](https://support.google.com/youtube/answer/72857#apply)。

您可以利用您视频最后 5-20 秒期间出现的片尾画面，宣传 (i) [与您的频道相关联且已通过验证的网站](https://support.google.com/youtube/answer/2887282)；以及 (ii) 获准网站上的商品和众筹活动。了解如何[添加片尾画面](https://support.google.com/youtube/answer/6388789)。
