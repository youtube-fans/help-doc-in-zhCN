# 管理您的权利

本部分的文章主要面向使用 YouTube [Content ID](http://www.youtube.com/t/contentid) 匹配系统的合作伙伴。您通过[上传内容](https://support.google.com/youtube/topic/21002)至内容管理系统来创建[资产](https://support.google.com/youtube/answer/3011552)，每项资产均代表着版权作品（例如录音、音乐视频或乐曲等）中的权利。接着，含有版权作品的视频会通过[版权声明](https://support.google.com/youtube/answer/3311596)（声明拥有视频内容的所有权）与各项资产关联起来。版权声明会将视频与特定的资产关联起来，并遵循资产所有者指定的[政策](https://support.google.com/youtube/answer/107383)（获利、禁播、跟踪）。
