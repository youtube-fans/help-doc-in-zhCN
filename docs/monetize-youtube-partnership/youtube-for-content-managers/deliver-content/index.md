# 提交内容

本部分的文章主要面向使用 YouTube 内容管理系统管理其版权内容的合作伙伴。您通过[上传内容](https://support.google.com/youtube/topic/21002)至内容管理系统来创建[资产](https://support.google.com/youtube/answer/3011552)，每项资产均代表着版权作品（例如录音、音乐视频或乐曲等）中的权利。

如果您启用了 [Content ID](https://support.google.com/youtube/topic/6186113) 匹配功能，YouTube 会将已上传至 YouTube 的视频与资产的[参考文件](https://support.google.com/youtube/answer/107004)进行对比，看是否存在版权冲突等问题。
