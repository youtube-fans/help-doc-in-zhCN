# 修改播放列表

您可以为播放列表添加或修改标题和说明、重新排序视频或从播放列表中移除视频。播放列表菜单中有两个标签：

* **基本标签** ：在此标签中，您可以更新隐私设置、控制新视频加入播放列表的方式、允许嵌入播放列表、调整播放列表的默认排序方式（注意，这会覆盖您对播放列表中的视频所做的任何手动排序），或者删除播放列表。
* **“自动添加”标签** ：您可以在此标签中定义一些规则，让系统根据视频所应用的标签或者视频标题或说明中的关键字，自动将视频添加到此播放列表中。一个播放列表可以应用多条规则。定义之后，我们会将规则应用于所有新上传的视频。如果上传的视频带有您指定的标签，或者视频说明或标题中含有您指定的关键字，该视频将会自动添加到您的播放列表中。

计算机

## 添加播放列表说明

1. 在计算机上，从导视面板中选择一个播放列表
2. 点击 **添加说明** ，然后输入播放列表详细信息
3. 点击 **完成**

## 修改播放列表标题或说明

1. 在计算机上，从导视面板中选择一个播放列表
2. 点击播放列表标题或说明，然后根据需要进行修改
3. 点击 **完成**

## 重新排序播放列表中的视频

1. 选择一个播放列表。
2. 将鼠标悬停在视频上，直到视频缩略图旁边出现灰色竖线。
3. 上下拖放视频，以重新排列播放列表中的视频。

## 从播放列表中移除视频

1. 打开一个播放列表。
2. 将鼠标悬停在要移除的视频上。
3. 点击视频标题右侧的“X”图标。

如果您无法按照这些说明操作，则您可能使用的是旧版 YouTube。如果您使用的是新版浏览器，请更新为[最新版本的 YouTube](https://www.youtube.com/new)。

**提示** ：您还可以在移动设备上使用 YouTube 工作室应用来执行此操作。如需了解入门信息，欢迎访问 [YouTube 工作室应用帮助中心](https://support.google.com/youtubecreatorstudio#topic=6048525)。
